# DeepSeek AI 助手集成 - 使用说明

## 功能概述

已为远程服务器管理器集成 DeepSeek AI 工具，支持多轮对话，并可在同一个面板里完成命令生成/解释与文件查找定位等任务。

### 核心功能

1. **💬 多轮对话**
   - 支持连续追问与补充信息
   - 支持把终端输出/文件管理信息按需作为上下文发送给 AI（可选）

2. **🤖 命令生成**
   - 用自然语言描述需求，AI 自动生成 Shell 命令
   - 示例："查找并删除3天前的日志文件" → `find /var/log -name "*.log" -mtime +3 -exec rm {} \;`

3. **📚 命令解释**
   - 解释任意 Shell 命令的作用和参数
   - 分析命令输出（stdout/stderr），提供错误诊断

4. **🔍 文件查找/定位**
   - 询问“某类配置/日志/数据文件一般在哪”，给出常见路径与查找命令
   - 示例："nginx 的配置文件在哪" → 常见路径 + `find/locate/grep` 命令

5. **⚡ 流式响应**
   - 实时打字机效果，边生成边显示
   - 无需等待完整响应

6. **💡 智能提示**
   - 提供常用命令快捷按钮
   - 一键使用常见场景命令

## 配置步骤

### 1. 获取 DeepSeek API Key

访问 [DeepSeek 开放平台](https://platform.deepseek.com/api_keys) 注册账号并获取 API Key。

### 2. 配置 API Key

- 在应用主界面点击 ⚙️ AI 设置按钮（位于主题选择器旁边）
- 填写 API Key 和选择模型
- 启用 AI 助手功能
- 点击保存

**配置选项说明：**
- **API Key**: 从 DeepSeek 平台获取的密钥（格式：sk-xxxxxxxx）
- **API 地址**: 默认 `https://api.deepseek.com`，一般不需要修改
- **模型选择**:
  - `deepseek-chat`: 通用对话模型，适合日常命令生成
  - `deepseek-coder`: 代码专用模型，适合复杂脚本生成
- **启用 AI 助手**: 勾选以启用所有 AI 功能

### 3. 开始使用

配置完成后，在服务器详情页的“终端工具”页签中：

1. 点击“显示 AI 工具”
2. 在对话框中输入问题并发送（支持多轮追问）
3. AI 回复中若包含可执行命令，可点击“执行命令”将命令发送到终端面板
4. 可按需勾选“包含终端历史 / 包含文件管理信息”让 AI 基于当前状态回答（注意会发送给 DeepSeek）

**命令解释：**
- 打开 AI 工具后，终端历史中每条命令旁会出现“解释”按钮，点击后会把命令与输出带入对话并生成解释与排查建议。

## 使用示例

### 场景 1：生成复杂命令

**输入**: "查看占用8080端口的进程"

**AI 生成**: `lsof -ti:8080 | xargs kill -9`

### 场景 2：解释陌生命令

点击任意已执行命令旁的"解释"按钮，AI 将解释该命令的作用。

**示例命令**: `lsof -i :8080`

**AI 解释**:
```
lsof -i :8080

- lsof: list open files（列出打开的文件）
- -i :8080: 筛选占用 8080 端口的网络连接

这个命令用于查看哪个进程占用了 8080 端口。
```

### 场景 3：错误诊断

执行命令出错时，点击"解释"按钮，AI 将分析错误原因并提供解决方案。

**错误示例**:
```
rm: cannot remove '/var/log/app.log': Permission denied
```

**AI 分析**:
```
错误原因：
- 当前用户没有删除该文件的权限

解决方案：
1. 使用 sudo 执行: sudo rm /var/log/app.log
2. 检查文件权限: ls -l /var/log/app.log
3. 切换到 root 用户: su - root
```

## 技术实现

### 新增文件

1. **electron/services/aiService.js** - AI 服务封装
   - DeepSeek API 调用
   - 流式响应处理
   - 多轮对话（可选上下文）/命令生成/解释/文件定位

2. **src/components/AIConfig.vue** - 配置组件
   - API Key 设置界面
   - 模型选择
   - 启用/禁用 AI 功能

3. **src/components/TerminalPanel.vue** (修改)
   - 集成统一 AI 工具（多轮对话）
   - 命令解释：点击历史命令旁“解释”
   - 文件定位：在同一对话中直接提问
   - 可按需注入文件管理/终端上下文

- ✅ 流式响应支持（Server-Sent Events）

## 注意事项

1. **API 费用**: DeepSeek API 按 token 计费，请注意使用频率
2. **网络连接**: AI 功能需要联网才能使用
3. **数据安全**: API Key 存储在本地，请妥善保管
4. **默认禁用**: AI 功能默认关闭，需要手动启用

## 故障排除

### 问题：AI 助手无法使用

**原因**: API Key 未配置或无效

**解决方案**:
1. 检查 API Key 是否正确
2. 确认网络连接正常
3. 查看浏览器控制台是否有错误信息

### 问题：流式响应卡顿

**原因**: 网络延迟或 API 响应慢

**解决方案**:
1. 检查网络连接
2. 等待响应完成，不要重复点击
